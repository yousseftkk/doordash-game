# 🚪 DooR DasH: Scare vs Laugh Touchdown ⚡

A competitive turn-based board game built in **Java**, where players race across a chaotic **100-cell factory floor**, balancing **movement, positioning, and energy management** to reach victory.

Inspired by the fast-paced energy battle theme of Monsters Inc.

---

## 🎮 Gameplay

Players take turns moving across a **10×10 zigzag board**, collecting energy while navigating hazards and disrupting opponents.

Winning isn't only about speed — smart positioning, timing, and action card usage can completely change the outcome.

---

## ✨ Features

### 🎲 Turn-Based Movement
- Dice-based movement system
- Re-roll mechanics triggered by spatial collisions
- Strategic positioning against opponents

### 🗺️ Dynamic Board System
- 10×10 zigzag floor layout
- Wrap-around movement mechanics
- Interactive cell behavior

### ⚠️ Special Tile Effects
Encounter different board events including:
- 🚪 Energy Doors
- 🔄 Conveyors
- 🧦 Contamination Socks

### 🃏 Action Card System
Use special cards to create chaos:
- 🔁 Position Swap
- ⚡ Energy Steal
- 🛡️ Shields
- 🎭 Role Confusion
